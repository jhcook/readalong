## ADR‑002: Use Whisper.cpp for Local Speech‑to‑Text
Status: Accepted

Context
We need local STT that:

Runs on desktop and mobile

Works offline

Has high accuracy

Is open‑source and maintainable

Decision
Use Whisper.cpp as the primary local STT engine.

Consequences
Positive

Cross‑platform

High accuracy

No cloud dependency

Efficient on CPU

Negative

Larger model sizes

Requires native bindings on mobile
