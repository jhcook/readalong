## ADR‑004: Use ElevenLabs API for Cloned Voice Narration
Status: Accepted

Context
We need high‑quality TTS in a parent’s cloned voice.
Local TTS cannot match ElevenLabs quality.

Decision
Integrate ElevenLabs TTS API for:

Cloned voice generation

Streaming TTS

Character voices

Consequences
Positive

Best‑in‑class voice quality

Streaming reduces wait time

Supports multiple voices

Negative

Requires internet

API cost considerations
