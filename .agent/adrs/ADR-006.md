## ADR‑006: Use React for Web UI and React Native for Mobile
Status: Accepted

Context
We need:

Shared UI logic

Cross‑platform consistency

Strong ecosystem support

Decision
Use:

React for web and browser extension UI

React Native for mobile apps

Consequences
Positive

Shared components and logic

Strong developer ecosystem

Easy integration with native modules

Negative

Requires native modules for Whisper.cpp

Performance tuning needed for animations
