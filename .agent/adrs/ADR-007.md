## ADR‑007: Use IndexedDB and Secure Mobile Storage
Status: Accepted

Context
We need to store:

Alignment maps

Cached audio

Reading progress

Settings

Storage must be:

Local

Secure

Cross‑platform

Decision
Use:

IndexedDB for browser

Keychain (iOS) and Keystore (Android) for mobile

Consequences
Positive

Secure storage

Works offline

Supports large audio files

Negative

Requires abstraction layer

IndexedDB quirks across browsers
