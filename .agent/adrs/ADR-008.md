## ADR‑008: Use a Local‑Only Privacy Model by Default
Status: Accepted

Context
The system is designed for children.
Privacy is non‑negotiable.
Parents must trust the platform.

Decision
Default to local‑only processing for:

Voice capture

STT

Alignment

Highlighting

Cloud features are opt‑in.

Consequences
Positive

Strong privacy guarantees

COPPA‑aligned

No accidental data leakage

Negative

Some features unavailable offline

More complex onboarding
