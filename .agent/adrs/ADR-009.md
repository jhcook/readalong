## ADR‑009: Use a Unified Alignment Map Format
Status: Accepted

Context
We need a consistent format for:

Word timestamps

Sentence boundaries

Confidence scores

Playback navigation

Decision
Define a JSON‑based alignment map with:

sentences[]

words[]

start, end timestamps

confidence

Consequences
Positive

Easy to debug

Works across platforms

Supports future enhancements

Negative

Requires versioning strategy
