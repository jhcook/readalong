## ADR‑010: Use a Child‑Safe UI Mode
Status: Accepted

Context
Children require:

Simple UI

Restricted access

Safety controls

Decision
Implement a Child Mode with:

Minimal UI

Large touch targets

PIN‑locked parent settings

No external links

Consequences
Positive

Safe for children

Reduces cognitive load

Prevents accidental changes

Negative

Requires dual UI flows

Additional testing
