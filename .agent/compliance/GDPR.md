# GDPR Compliance Checklist

## Data Minimization
- [ ] We only collect data that is strictly necessary.
- [ ] PII is not logged to console or text files.

## Right to Erasure
- [ ] There is a mechanism to delete this user's data.
- [ ] Backups are handled (or at least documented) regarding deletion.

## Consent
- [ ] Users explicitly consent to data collection.
- [ ] Cookie banners are present if tracking is used.

## Security
- [ ] Data at rest is encrypted.
- [ ] Data in transit is encrypted (TLS).
